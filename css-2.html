盒模型

每个 HTML 元素都是一个盒子。从内到外：内容 → padding → border → margin。

┌────────── margin ──────────┐ │ ┌──────── border ────────┐ │ │ │ ┌───── padding ─────┐ │ │ │ │ │ │ │ │ │ │ │ 你的内容在这 │ │ │ │ │ │ │ │ │ │ │ └────────────────────┘ │ │ │ └─────────────────────────┘ │ └──────────────────────────────┘
